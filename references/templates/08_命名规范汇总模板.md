# 08_命名规范汇总模板

## 引擎分支

| 引擎/工具 | 命名证据 |
|---|---|
| Unity | asset name、path、namespace、class |
| Unreal | asset path、Blueprint、Material、Texture 前缀 |
| GameMaker | sprite/object/script/sound 命名 |
| 未知 | 文件名、目录名、扩展名 |

## 必填分析细节大纲

1. 资源前缀：UI、角色、场景、道具、音频、VFX、Shader。
2. 动作命名：角色、动作、方向、帧号。
3. 代码命名：namespace、class、system、manager、definition。
4. 路径层级：项目目录、工具目录、可读资源目录。
5. 我方项目命名建议。

## 文档模板

```markdown
# 08_命名规范汇总

## 资源命名

## 动作命名

## 代码命名

## 我方建议
```
