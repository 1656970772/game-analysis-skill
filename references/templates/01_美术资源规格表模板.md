# 01_美术资源规格表模板

## 引擎分支

| 引擎/工具 | 规格证据 |
|---|---|
| Unity | Sprite、Texture2D、SpriteAtlas、AnimationClip |
| Unreal | Texture、Material、Paper2D、Widget |
| GameMaker | texture page、sprite、background、font |
| 未知 | 图片尺寸、目录结构、文件名 |

## 必填分析细节大纲

1. 分辨率分布：按 `≤64px`、`65-256px`、`257-512px`、`513-1024px`、`≥1025px` 汇总。
2. 图集规格：常见 256/512/1024/2048 尺寸、是否混合 UI/角色/场景。
3. UI 图标规格：常见宽高、比例、透明通道。
4. 序列帧规格：角色、动作、帧尺寸、帧数。
5. 对标表：参考项目规格 vs 我方项目建议规格。
