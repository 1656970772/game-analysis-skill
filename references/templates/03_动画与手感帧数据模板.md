# 03_动画与手感帧数据模板

## 引擎分支

| 引擎/工具 | 动画证据 |
|---|---|
| Unity | AnimationClip、AnimatorController、Sprite 序列帧 |
| Unreal | AnimationSequence、Flipbook、Niagara |
| GameMaker | sprite frames、object step/draw events |
| 未知 | 文件名序列、尺寸和动作命名 |

## 必填分析细节大纲

1. 动作清单：角色、动作名、帧数、资源路径。
2. 帧数据表：前摇、有效帧、后摇、取消窗口，无法确认时标为待验证。
3. 反馈层：残影、粒子、音效、暂停、震屏。
4. 命名规则：动作名、帧号、方向、变体。
5. 对标调参：用于原创项目的帧数建议。
