# 02_美术风格与色彩速查模板

## 引擎分支

| 引擎/工具 | 色彩证据 |
|---|---|
| Unity | Texture2D/Sprite、Material、Shader |
| Unreal | Texture、Material Instance、PostProcess |
| GameMaker | texture page、background、sprite |
| 未知 | 图片像素抽样、文件名与画廊 |

## 必填分析细节大纲

1. TOP 主色卡：hex、出现占比、样本类别。
2. 明度层级：背景、角色、UI、危险提示的亮暗关系。
3. 饱和度策略：角色与环境、前景与后景、交互物与装饰物。
4. 风格语法：线条、边缘、材质、形状、UI 边框。
5. 原创化改写：保留可读性规律，不复用原图、角色、图标或构图。
