# 09_可迁移技术清单模板

## 引擎分支

| 引擎/工具 | 可迁移证据 |
|---|---|
| Unity | 资源规格、代码架构、场景、动画、渲染索引 |
| Unreal | 资源路径、Blueprint、Material、Level |
| GameMaker | sprite、object、room、script |
| 未知 | 文件结构和可读报告 |

## 必填分析细节大纲

1. P0：立即可用的规格/流程，不复用原始资源。
2. P1：需要小样本验证的系统、工具或规范。
3. P2：依赖美术试画、手感调参或性能验证的方向。
4. 工时预估：半天、1-2 天、3-5 天、一周以上。
5. 验收标准：原创资源、原创代码、可运行原型或团队规范。
